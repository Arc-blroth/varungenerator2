# Varun Generator 2
### The Digital Toaster

Varun Generator 2 is a simple webapp that matches nouns with alliterative adjectives.


It interfaces [NLTK](http://www.nltk.org/) with an [express.js](https://expressjs.com/)
backend that serves a vanilla HTML5 webpage.
